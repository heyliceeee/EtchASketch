# Etch-a-Sketch  
Um mini‑projeto em Python usando o módulo **turtle**, onde controlas a tartaruga com o teclado para desenhar no ecrã — como uma versão simples do clássico *Etch‑a‑Sketch*.

## 🎯 Objetivo  
Permitir ao utilizador mover a tartaruga em quatro direções usando as teclas **W, A, S, D**, e limpar o desenho com a tecla **C**.

---

## 🧠 Funcionalidades  
- **Mover para a frente** → tecla `W`  
- **Mover para trás** → tecla `S`  
- **Rodar para a esquerda** → tecla `A`  
- **Rodar para a direita** → tecla `D`  
- **Limpar o desenho e voltar ao centro** → tecla `C`  

A janela fecha apenas quando clicas nela.
